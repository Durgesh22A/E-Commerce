# 🛒 Modern E-Commerce Store

A fully functional, responsive, and high-performance E-Commerce application built with **React.js**, **Vite**, and **Framer Motion**. This project features real-time data fetching from the DummyJSON API and includes advanced features like pagination, category filtering, and a persistent cart system.

## 🚀 Live Demo
Check out the live website: [**https://e-commerce-durgesh.vercel.app**](https://e-commerce-durgesh.vercel.app)

## ✨ Key Features
* **Dynamic Product Listing:** Fetches 190+ products across various categories.
* **Advanced Pagination:** Smooth 12-item per page navigation.
* **Smart Filtering:** Category-based filtering and search functionality.
* **Persistent Cart & Wishlist:** Uses Context API and LocalStorage to save your items.
* **Interactive UI:** Modern animations using Framer Motion and toast notifications with React-Toastify.
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views.
* **Clean Routing:** Multi-page experience with React Router 6.

## 🛠️ Tech Stack
* **Frontend:** React.js (Vite)
* **State Management:** React Context API
* **Routing:** React Router DOM
* **Styling:** Modern CSS3 & Framer Motion
* **API:** DummyJSON API
* **Deployment:** Vercel
